I am a young programmer currently persuing an Ontario high school diploma with the ILC. I have participated in FIRST (For Inspiration and Recognition of Science and Technology) Robotics for five years.

My primary development environment is Ruby on OS X Lion with Ruby on Rails as my preferred web framework. I'm known to delve into Python, C++, PHP, Haskell and JavaScript.

This website is made with Jekyll using Markdown, Pygmentize and Sass. Source code is available at GitHub.
